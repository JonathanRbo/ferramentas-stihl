# STIHL - Landing Page de Ferramentas

## Tecnologias Utilizadas
- HTML5
- CSS3 (Variaveis CSS, Animacoes Keyframes, Backdrop Filter)
- JavaScript (Contador animado, Smooth scroll, Embla Carousel)
- Squeleton CSS Framework
- WOW.js (Animacoes de scroll)
- Google Fonts (Inter)

## Funcionalidades Principais
- **Header Sticky**: Navegacao fixa com efeito glassmorphism
- **Hero Section**: Secao inicial com background de imagem e efeito de brilho pulsante
- **Estatisticas**: Cards com numeros da empresa (95 anos, 160 paises, 18.000 funcionarios)
- **Cards de Beneficios**: 6 diferenciais da marca com icones animados
- **Catalogo de Produtos**: 6 produtos com preco e modal de detalhes
- **Carousel de Depoimentos**: Slider automatico com reviews de clientes
- **Secao CTA**: Call-to-action para contato
- **Menu Mobile**: Modal para navegacao em dispositivos moveis

## Produtos Apresentados
- MS 261 C-M (Motosserra)
- BR 600 MAGNUM (Soprador)
- FS 131 R (Rocadeira)
- HSA 66 (Tesoura)
- RE 130 PLUS (Lavadora)
- AP 300 S (Bateria)

## Design
- Paleta de cores: Laranja STIHL (#FF6600), Preto (#000000), Cinza escuro (#1a1a1a)
- Visual premium e profissional
- Efeitos de hover 3D nos cards
- Background com patterns SVG
